Notion URL Tracker — Settings

Profiles

Each profile is a separate Notion target (e.g. Work and Personal). The selected profile is the one used when saving; you can switch it from the popup too.

1. Connect Notion

Create an internal integration at notion.so/my-integrations, copy its Internal Integration Secret, then share a parent page with that integration (page → ••• → Connections).

2. Choose database

Using the Link Tracker template? Duplicate it into your Notion, share the page with your integration, then pick its Links database below — a Tags property is added automatically if missing.

Or create a brand-new database

Pick a parent page shared with the integration, then create an inline read-later database inside it.

Or paste a database ID

3. Predefined tags

One per line (or comma-separated). Shown as suggestions when saving; you can still add new tags on the fly.

Settings

Keyboard shortcuts

Ctrl/⌘+Shift+U opens the save popup. Ctrl/⌘+Shift+S instantly saves the current page with defaults. Change them at edge://extensions/shortcuts.
